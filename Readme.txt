Instrucciones:

1) Instalar Anaconda:(copiar y pegar en el navegador)
https://repo.anaconda.com/archive/Anaconda3-2019.03-Windows-x86_64.exe

2)En el instalador de anaconda, al principio les dira que si quieren agregar anaconda al path de windows, seleccionan esa opcion y continuar a todo lo demas.

3)En la carpeta del programa, primero instalar el ejecutable que dice "Install_Requirements"
	
4)Una vez instalado los requerimientos 
	Agregar las imagenes deseadas en la carpeta img_input
	Ejecutar el programa Run_Captcha y las imagenes saldran procesadas en 
	img_output.


Gracias por usar mi programa.
